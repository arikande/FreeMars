Engineer

The barren and inhospitable terrain of Mars would be of little use to settlers if there were not engineers. Using specifically designed equipment, these units do all kinds of improvements on Martian surface like roads, irrigation and mines.

This unit does not have any armor or armament and can not fight, it will be destroyed even if an enemy unit with minimum arms attacks it, thus the engineer must not be left undefended.


Movement points 1
Attack 0
Defence 0
Production cost 2000
Upkeep cost 70

Can move on:
  • Plains
  • Desert
  • Wasteland
  • Tundra
  • Crater
  • Hills
  • Mountains
