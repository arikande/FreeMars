Keyboard shortcuts

F1:Display help
F2:Display colony manager
F3:Display unit manager
F4:Display missions
F5:Display trade
F6:Display diplomacy
F8:Quicksave
F9:Quickload
Ctrl + Q:Quit game
Enter:End turn
C:Center on active unit
N:Activate next unit
Space:Skip current unit and activate next unit
Delete:Disband unit
B:Build colony
F:Fortify
S:Sentry
R:Build road
I:Build irrigation
V:Clear vegetation
M:Build mine
Ctrl + R:Rename unit
+:Main map zoom in
-:Main map zoom out
*:Main map default zoom
Ctrl + "+":Mini map zoom in
Ctrl + "-":Mini map zoom out
Ctrl + "*":Mini map default zoom
Ctrl + D:Toggle display tile coordinates
Ctrl + G:Toggle display tile grid
O:Display orbit
E:Display Earth
Ctrl + O:Send unit to orbit
Ctrl + E:Send unit to Earth
Ctrl + U:Unload all cargo
Ctrl + T:Display colony tax dialog
